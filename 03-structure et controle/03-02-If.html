JS - III 02 IF

1- If ...

La condition if est l'une des plus utilisées et est également la plus simple à appréhender puisqu'elle vas nous permettre d'éxecuter un bloc de code Si et SEULEMENT SI le résultat vaut true

2- If exécuter dans le if.js

  • if(x > 1){
    alert('x contient une valeur supérieure a 1');
  • if(x == y){
    alert('x et y contient la même valeur');
  • if(y){
    alert('La valeur de y est évalué a true');

! ! dans ce 3eme if : tout test d'une ondition va être évalué dans un contexte booléen ; donc a true à l'exception des valeurs suivantes qui vont être évaluées à false

  • Le booléen false
  • La valeur 0
  • Une chaîne de caractère vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)

2- Inverser la valeur logique d'un test

Dans un if le code placé dans le if n'est éxécuté aue si le résultat de la comparaison est true

Dans certaines situation, on préfère vérifier si la comparaison est false

On peut vérifier que c'est faux en comparant explicitement le résultat de la comparaison

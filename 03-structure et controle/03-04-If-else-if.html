JS - III 04 IF ELSE if else

1- If...else ...if...else ....

La condition if...else... if...else... est un structurre conditionelle encore plus complete puisqu'elle permet de généerer et de prendre en charge autant de cas que l'on souhaite

Nous allons pouvoir écrire autant de else if que l'on veut

Dès qu'un test va être validé le JS va ignorer les tests suivants. Il faut donc faire bien attention à l'ordre des else if lors de l'écrtiture d'une condition pour obtenir le résultat souhaité

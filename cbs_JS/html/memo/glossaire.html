Glossaire

Assignation
Assigner une variable et lui attribuer une valeur
Ex : en JS, on utilise le symbole =
​
Binaire
Fichier lisible et exécutable par l'ordinateur
​
Caster
Opération de conversion d'un type de données dans un autre
​
Chaine vide
un caractère signifiant qu'une chaîne est vide saut de ligne : un caractère signifiant que le logiciel doit afficher un retour à la ligne
​
Compiler
Analyser un code et fabriquer un binaire
​
Concaténer
Rabouter 2 chaînes de caractère en une seule
​
Console
Zone permettant de consulter les messages produits par JS
​
Déclaration
Action de créer quelque chose en programmation : concernera souvent les variables
​
Echapper des caractères
Enlever sa fonction à un caractère afin de simplement l'afficher
​
Instruction
Unité de code à lancer; fonctionne souvent ligne par ligne
​
Interpréter
Analyser un code et le faire exécuter à l'ordinateur
​
Mot-clé
Un mot-cé est un mot "compris" ou ayant un sens en JS : let
​
Saut de ligne
Un caractère signifiant que e logiciel doit afficher un retour à la ligne
​
Syntaxe
Convention d'écriture d'un code informatique. JS va utiliser un certains nombre de symbole et de mot clé et de règle d'écriture du code, ces conventions s'apellent syntaxe du langage
​
Variable
Elément de programmation qui va servir à contenir quelque chose : nombre, une chaîne, etc, et sera enregistré au moyen d'un nom
ex : let bidule
